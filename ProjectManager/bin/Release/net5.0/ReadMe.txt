1. Запустить приложение, запустив файл ProjectManager.exe
2. Изначально, программа выдаст ошибку, т.к. файл congig.cfg отсутсвует.
   При этом программа запросит "Хотители вы ввести путь к базе данных вручную.
   Введите с клавиатуры "y" для ввода пути к базе данных в ручную, или "n", если не хотите вводить путь к
   базе данных.
   Если файл congig.cfg существует, и путь к базе данных правильный, программа запросит имя пользователя
   и пароль.
3. В случае, если программа не обнаружит базу данных по указаному пути, либо отказа пользователя от ввода
   пути к базе данных, программа предложит создать новую базу данных.
   Введите с клавиатуры "y" для создания новой базы данных, или "n", если не создавать новую базу данных.
4. Если пользователь решил создать новую базу данных, программа запросить имя базы данных. Имя базы данных
   является полным путём к расположению этой базы данных.
5. Т.к. база данных пуста, для работы с базой данных необходимо создать первого пользователя, который
   будет и администратором базы данных. Введите имя пользовтеля.
6. Введите пароль.
7. После создания первого пользователя, программа запросит команду. Для ознакомления со списком комманд
   необходимо ввести команду show commands.
8. Для ввода команд,необходимо прописать команду текстом. Некоторые команды требуют прав администратора.
   Вводите команды, и следите за запросами программы, для их исполнения
9. Для выхода из программы необходимо ввесит "q" или "Q".
   При этом программа спросит у пользователя о сохранении изменений, если такие были, и необходимости
   установить выбранную базу данных как базу данных по умолчанию, если изначальная загрузка базы данных
   была не успешной, либо в ручную. В случае положительного ответа программа создаст или перезапишет файл
   congig.cfg с указанием пути до базы данных.
